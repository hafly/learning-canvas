# canvas-test

学习canvas的一些例子<br>
并写了一个canvas的绘图工具，涉及3D空间运动计算，3D坐标转屏幕坐标。

## 安装依赖
```
npm install
```

## 构建menu列表
```
npm run build
```

## 安装http-server，运行服务
```
http-server
open http://127.0.0.1:8080/menu.html
```
